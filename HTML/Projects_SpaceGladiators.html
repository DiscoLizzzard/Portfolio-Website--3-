Space Gladiators

Play the Game on Itch.io

Description

Space Gladiators is inspired by fighting games like Super Mario Bros, but with guns. Captured by alien forces, players must compete in gladatorial combat to earn their freedom Space Gladiators is a competitive multiplayer game with a strong emphasis on local-play.

Features

  • Shoot powerful lazers to thrust yourself forward!
  • Players have limited energy, which provides power to both weapon and life support. Don't waste it!
  • Project created solo for the theme "Unknown Worlds"
